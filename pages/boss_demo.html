Boss Demo

Summer 2025
Role: Creator
Tools: Unity, C#
Platform: PC

A small demo boss battle. This project is still in development.

I did this project to practice the Hierarchical State Machine game programming pattern. To do this this I needed to design an enemy AI with complex enough behavior to warrent this pattern. I ended up with a 3 phase boss battle, with the 3 phases being root states and the behavior or the phases being the sub states.
